Veja como funciona o Dropdown

Alterne sobreposições contextuais para exibir listas de links e muito mais com o plug-in suspenso do Bootstrap. Os menus suspensos são sobreposições contextuais, para exibir listas de links e muito mais. Eles são interativos com o plug-in JavaScript do Dropstrap incluído. Eles são alternados clicando, não pairando; esta é uma decisão de projeto intencional.Dropdowns são construídos em uma biblioteca de terceiros, Popper.js , que fornece posicionamento dinâmico e detecção de viewport.

Botão Divisão

Da mesma forma, crie menus suspensos de botões divididos com praticamente a mesma marcação que os botões suspensos de botão único, mas com a adição de .dropdown-toggle-splitespaçamento adequado ao redor do cursor suspenso. Usamos essa classe extra para reduzir a horizontal paddingem ambos os lados do cursor em 25% e remover o margin-leftque é adicionado para suspensões de botão regulares. Essas alterações extras mantêm o cursor centralizado no botão dividido e fornecem uma área de acerto com tamanho mais adequado ao lado do botão principal.
